    Instructions

    The purpose of CodeFusionX is to gather various programming files and combine the code into a single text, making it easier to send to a large language model.

    1. Click the "Select File" button to choose individual files, or click the "Select Directory" button to select an entire directory containing programming files.
    2. The selected file paths will be displayed in the sidebar.
    3. The contents of the selected files will be combined and displayed in the textbox.
    4. To include the file name and language-specific comments in the combined text, keep the "Include File Name" checkbox checked.
    5. To copy the combined text to the clipboard, click the "Copy to Clipboard" button.
    6. To export the combined text as a text file, click the "Export to Text File" button.
    7. To clear the textbox and file paths, click the "Clear" button.
    8. To toggle between light and dark modes, use the "Dark Mode" checkbox.
